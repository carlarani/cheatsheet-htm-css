Cheatsheet: HTML and CSS

HTML

HTML é constituída de tag que marcam o que cada elemento da página é. É importante aplicar tags semânticas (explicitam a função), sempre que possível. Cada tag, no geral, é composta de uma tag de abertura, o conteúdo em si e uma tag de fechamento.

CSS

CSS é cascateamento de marcações de estilo. Pode ser feito com base em tags, classes ou ids.
CSS pode ser descrito inline no próprio arquivo html (não recomendado) através do atributo style diretamente nas tags, ou até em um bloco prórpio na tag style dentro do A alternativa mais recomendada, entretanto, é manter o arquivo de estilo completamente separado do conteúdo. Sendo assim, o CSS vai em um rquivo proprio e deve ser conectado ao arquivo html através da tag link no head.

<html>
<head>
<link rel="stylesheet" href="styles.css">
</head> </html>
